short UndertaleModLib.IpcMessage_t.FuncID

A numerical value (1 to 4 for the latest AUMI implementation).

The following table describes and explains the numerical values:

number Explanation
1 Test Communication - Always returns a string that's 128 characters long.
2 Get Function By Index - Returns information about a function at a specified index in the runner.
3 Get Function By Name - Returns information about a function with a specified name.
4 Execute Code - Executes precompiled bytecode in the global context.
